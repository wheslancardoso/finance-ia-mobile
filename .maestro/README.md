# Maestro E2E Tests para Vesper Finance Mobile
# Pré-requisitos:
#   1. Instalar Maestro CLI: curl -Ls "https://get.maestro.mobile.dev" | bash
#   2. Rodar o app em um emulador/simulator via Development Build (não Expo Go)
#   3. Executar: maestro test .maestro/

# Notas:
# - Maestro usa seletores visuais (texto, testID, etc.) sem precisar instrumentar o app
# - Fluxos declarativos em YAML — legíveis e fáceis de manter
# - Para usar com Expo Go: maestro não suporta, é necessário um dev build
